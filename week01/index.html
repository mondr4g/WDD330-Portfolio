Syllabus

Course Objectives

Demonstrate knowledge and skills to create dynamic web sites using:

  • Ptinciples introduced in Web Frontend Development.
  • PHP.
  • The Model-View-Control (MVC) design architecture.
  • MySQL database(s) and SQL.
  • Data validation.
  • Problem solving through independent study, working effectively with others, communicating clearly, fulfilling assignments and meeting deadlines.

Grading

The grade is based on the demonstrated competency of each course objective. Each assessment is tied and scored directly to one or more course objectives.

The grade in the course is based on the number of course objectives of which we have demonstrated mastery. Passing course objectives, not the gross percentage of the class, determines the final course grade. Each objective is graded Pass / Fail. To pass an objective we must earn at least 80% of the points allocated to it.
